# ReadToMe

Chrome extension for reading selected text aloud using Google Cloud Text-to-Speech Chirp voices.

## Setup

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select this repository folder.
4. Open the extension **Options** page.
5. Set your Google Cloud API key and preferred Chirp playback settings.

## Usage

1. Select text on any web page.
2. Right-click and choose **Read selected text aloud (Google Chirp)**.

The extension sends selected text to Google Cloud Text-to-Speech and plays the synthesized audio.
